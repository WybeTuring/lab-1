The Jamestown project

Together with a couple of friends, we visited Jamestown (Accra, Ghana) and interacted with the residents of this historic city to better understand their culture and ways of live. The long term goal was to find ways in which these people could benefit more from the huge number of tourists that visist Jamestown each year, while at thesame time making the tourist experience world class. I did like this project, but sometimes the long days under the Accra sun were dreadful.

3

Library Management System

Using Matplotlib and Python, I built a library management system that could be run with much ease from the command line. The system allowed Library administrators to register in books in several ways, thereby making the task easier. For example, a librarian could register a new book by typing out the details in an interactive promppt, or by feeding an excel sheet as input. Matplotlib also enabled me to include some basic data analysis with beautiful graphs.

Two major challenges faced during this project included;

  • The teacher did not give me my full marks, as I forgot to work in a local python environment...Sad
  • I lost the code for this project afterwards. I should rebuild it someday

Airflight Route Optimizer

I fell in love with Edsgar W. Dijkstra by reading about his many idiosyncracies on Wikipedia and I was so excited when I decided to implement his algorithm as part of a project I was working on with the great Percy Brown.
